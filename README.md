# Além dos Olhos

Este projeto simples que consite em pegar as fotos do concurso fotográfico "UFMS Além dos Olhos", dispnível no [repositório institucional da universidade](https://repositorio.ufms.br/handle/123456789/3281), e exibí-las como uma galeria de fotos. Logo no início há a exibição das 13 fotos que compôem o calendário.

O código gira em torno do scrapping dos itens no site, seguido de outra busca em cada item para adquirir as informações da imagem, além de seu url para visualização. Por fim estas informações são salvas em cache de memória para melhorar significativamente a resposta ao usuário.
